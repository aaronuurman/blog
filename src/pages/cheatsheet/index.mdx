## 5. Visual Studio Code

### 5.1. Generate installed extensions list

Copy and paste outcome to somewhere safe for backup.

```bash
code --list-extensions | xargs -L 1 echo code --install-extension
```

### 5.2. Make sure terminal opens always right

Add to settings.

```bash
"workbench.panel.defaultLocation": "right"
```

## 6. Linux

### 6.1. Get build info

```bash
uname -a
# Linux dark-side 5.6.14-300.fc32.x86_64 #1 SMP Wed May 20 20:47:32 UTC 2020 
# x86_64 x86_64 x86_64 GNU/Linux
```

### 6.2. Get kernel version

```bash
uname -r
# 5.6.14-300.fc32.x86_64
```

### 6.3. See what services are available

```bash
ll /etc/systemd/system
```

### 6.4. See what services are running

```bash
journalctl -ef
```
